Slider


Usage

Link to an image using the following html syntax:


  <div id="slider>
    <img src="URL to image" />
    <img src="URL to image"/>
  </div>

Example on a call with default settings:


  $(document).ready(function(){
    $('#slider').slider({
    });
  });
  

  The default settings are
      transition: 'fade',
      time:     '4000',
      width: original width of the image,
      hight: original hight of the image

The slider example above uses the following customized settings:


  $(document).ready(function(){
    $('#slider').slider({
      transition: 'fade',
    time:       '3000',
    width:      '940',
    hight:      '300'
    });
  });



Lightbox



Usage

Link to an image using the following html syntax:


  <a class="lightbox" href="Url to image">
    <img src="Url to image" />
  </a>

Include the following css syntax for styling lightboxes.


  #overlay {
    position:fixed;
    top:0;
    left:0;
    height:100%;
    width:100%;
    background:black url(loader.gif) no-repeat scroll center center;
  }
  
  #lightbox {
    position:fixed;
  }

Example on a call with default settings:


  $(document).ready(function(){
    $('.lightbox').lightbox({
    });
  });
  

  The default settings are
      opacity: '0.5',
      color: 'black',
      width: original width of the image,
      hight: original hight of the image

The lightbox examples above uses the following customized settings:


  $(document).ready(function(){
    $('.lightbox').lightbox({
    opacity: '0.8',
    color: '#551F0D',
    width: '700',
    hight: '500'
    });
  });



Gallery


Usage

Link to an image using the following html syntax:


  <div id="gallery">
    <div class="gallery-current"><img/></div>
    <div class="gallery-all"> 
      <img src="img/abstract_autumn_background3.jpg"/>
      <img src="img/abstract_autumn_background4.jpg"/>
      <img src="img/abstract_autumn_background2.jpg"/>
      <img src="img/abstract_autumn_background1.jpg"/>
      <img src="img/abstract_autumn_background0.jpg"/>
    </div>
  </div>

Include the following css syntax for styling the gallery.


  .gallery-current img {
    margin: 0px 5px 0px 5px;
  } 
  .gallery-all {
    border: 1px solid #bbb;
    overflow: scroll;
  }

Example on a call with default settings:


  $(document).ready(function(){
    $('#gallery').gallery({
    });
  });
  

  The default settings are
      imageSide:       'right',
      imageWidth:      '600',
      imageHeight:     '400',
      thumbnailWidth:  '150',
      thumbnailHeight: '100'

The lightbox examples above uses the following customized settings:


  $(document).ready(function(){
    $('#gallery').gallery({
      imageSide:       'left',
      imageWidth:      '600',
      imageHeight:     '400',
      thumbnailWidth:  '150',
      thumbnailHeight: '100'
    });
  });
